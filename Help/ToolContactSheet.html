Create Contact Sheet Wizard

The contact sheet creation wizard creates contact sheets to display the images in the current folder. The contact sheet takes settings from the print view and can be accessed only from the print mode.

The wizard takes you through several steps and will prompt you for the following settings. For more information see How to create contact sheets.

The wizard has the following sections:

Input: Allows you to select files to be processed, either all or selected images. Images in sub folders can also be processed.

Contact Sheet Settings: Define the overall contact sheet size and other view options. You can enter or select the proposed contact sheet size. Thumbnail size is automatically calculated based on the contact sheet dimensions.

Other settings can be configured in the view options. These options control the geometry of a web page including the number of thumbs per column or row; as well as annotative text.

File Out: Defines either a folder where generated files will be output.

File Format: Selects the format of the completed contact sheet or sheets.
