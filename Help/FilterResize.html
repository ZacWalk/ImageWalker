Resize

Resize allows an image to be made larger or smaller by changing the number of pixels it contains. An image with less pixels fill take up less space when saved but will not be as detailed.

ImageWalker allows three main ways or types of resizing an image.

  1. Pixel based dimensions are entered as a number of pixels directly.
  2. Inch based dimensions are entered in inches and a new pixel size is generated based on image resolution (pixels per inch)
  3. Centimetres based dimensions are entered in centimetres and a new pixel size is generated based on image resolution (pixels per centimetres)

It is also possible to set a filter that will be used to resize the image. This filter defines how the pixels in the image will be sampled during the resize. Selecting Optimised will normally give the best results. Experimenting with different filters is also recommended because different images may respond better to a specific filter.
