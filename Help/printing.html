Printing Images

To print images you need to enter print mode. To enter print mode click print on the main toolbar.

The print mode will display a preview of what is to be printed on the left and a set of print settings on the right.

You can control the number of images printed on each page by changing the print or contact sheet dimensions in the view options. The print or contact sheet dimensions are found on the view layout tab. In the ImageWalker program press F4 to bring up the view options.

Troubleshoot printing

On some systems ImageWalker may hang during printing or print blank pages. Increasing the printer timeout setting can fix this problem. To do this:

  1. Under the Start/Settings menu in windows click on Printers.
  2. Right click on the printers icon and select properties.
  3. Select the Details Tab.
  4. Increase queue time or Timeout setting to from 15 to 45 seconds. (Transmission retry should already have a default value of 45 seconds)
