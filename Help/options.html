The Options Dialog

The view options dialog is used to configure the ImageWalker user interface as well as options for printing, contact sheet and web page creation.  It has various sections:

Thumbnail Size

This section allows the user to define the thumbnail size onscreen. If you change the current setting ImageWalker re-renders the current thumbnails.

This section also has an option to enable or disable special effects such as blending or fading of images being displayed in a slide show.

Thumbnail Annotations

Controls the text that appears under thumbnails in the normal thumbnails view.

Thumbnail Columns

Controls the columns that will be displaced in the thumbnails view when it is in detail mode.

Slideshow Annotations

Controls the annotations that will be seen below an image when it is shown in a slide show.

Behaviour

The Show images full screen when double clicked option allows images from the folder view to be shown full screen when double clicked.

Never obscure never allows thumbnails to be obscured in the folder view.

Show thumbnail tooltips allows a tool-tip popup window to be shown when the mouse pointer moves over a thumbnail.

Show EXIF, ICC and IPTC markers places small icons over thumbnails to show when they contain meta data.

Show hidden and system files allows you to show hidden files.

Automatically save images automatically saves images in the image view.

Use short dates allows short dates to be shown in all annotations.

Graphics

Options to allow you to turn off some of ImageWalker’s advanced graphical effects. This can help to solve display problems with ImageWalker.

Slide Show

Options that control what is shown during a slide show.

Image Capture

Options for the file format used to save images when captured from the screen, scanner or camera.
