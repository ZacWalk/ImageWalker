Registration of ImageWalker

ImageWalker is provided as freeware for private (non-commercial) or educational (including non-profit organization) use. If you like ImageWalker and want to contribute to the project you can optionally register for a small fee.

For commercial use, it is required to register this software. Registering is an important source of support in the development of future versions.

Instructions on how to register can be found on the ImageWalker web site at www.ImageWalker.com\register.html.
