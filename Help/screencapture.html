The Screen Capture Dialog

The Screen Capture Function lets you copy all or part of your computer screen into an image file. The new image files will be created  in the current folder.

ImageWalker captures multiple screen shots; you start capture mode; capture one or more images; then stop capture mode.

To start capturing click Start Screen Capture under the capture menu. You will be prompted with several capture options:

  1. Hotkey - This is the key combination that triggers a screen capture.
  2. What to capture - This is the area of the screen that will be captured.
  3. File output type - This is the format of the file the captured screen will be written to.
  4. Include cursor - This allows the current cursor to be in the capture.

To stop capturing screen shots click Stop Screen Capture under the capture menu.
