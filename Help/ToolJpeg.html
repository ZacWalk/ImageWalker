Loss-Less Jpeg Transformation Wizard

Loss-Less Jpeg Transformation allows multiple JPEG images to be rotated with ought quality loss. The wizard takes you through several steps and will prompt you for the following settings. 

Input: Allows you to select files to be processed, either all or selected images. Images in sub folders can also be processed.

Select Action: Define the transformation required, for example rotate or mirror.

Output: Defines where generated folders will go.

Note: In some cases Rotate No-Loss may remove pixels from the image boundary due to limitations in the way JPEG images are stored. Typically the rotate will be rounded off to the closest 8 pixel boundary.

Tip: If you only need to rotate a single image you can use the rotate no-loss filter. The crop no-loss filter will also allow cropping of JPEG images with no loss.
