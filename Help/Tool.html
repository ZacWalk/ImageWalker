Tools

ImageWalker supports many tools that perform batch processing operations. These tools normally take the form of a Wizard.

ImageWalker Tools:

  • Add a Keyword to Multiple Images Wizard
  • Animated Gif Slide Show Wizard
  • Batch Process Wizard
  • Contact Sheet Wizard
  • Convert Multiple Images Wizard
  • Loss-Less Jpeg Transformation Wizard
  • Rename Series Wizard
  • Web Page Wizard
