The Create Web Page Wizard

The web page creation wizard creates web pages or HTML files to display the images in the current folder. This wizard can only be run from the web view. The web view is where most of the look and feel settings are set that control the  Web Template and a Style Sheet used for the web page.

The wizard takes you through several steps and will prompt you for the following settings.

Input: Allows you to select files to be processed, either all or selected images. Images in sub folders can also be processed.

Thumbnail Size: The thumbnail size section allows the user to select the size and format of thumbnails in the completed web page.

Output: Defines the filename used for the web page and a location where it will be output to. A home URL can also be set that is used on some web pages that contain a home link. The advanced options are also available from this screen.

 

Tip: Web pages will incorporate descriptions entered for images.
