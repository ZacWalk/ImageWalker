Search

ImageWalker will search the current folder and subfolders for images or files that match the provided search criteria.

ImageWalker supports several logical operators to help specify a search:

  • OR
  • AND can be abbreviated as +
  • NOT can be abbreviated as -

An example might be to search for images with the keywords holiday and Malta type the search expression 'holiday AND Malta' into the Containing Keywords box.
