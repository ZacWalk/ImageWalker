Convert Multiple Images Wizard

The Convert Multiple Images wizard converts images in the current folder to a different format. The wizard takes you through several steps and will prompt you for the following settings.

Input: Allows you to select files to be processed, either all or selected images. Images in sub folders can also be processed.

File Format: Defines what format images will be converted to. For more information on file formats click here.

Output: Defines where generated folders will go.
