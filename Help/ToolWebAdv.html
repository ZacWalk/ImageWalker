The Create Web Page Wizard Advanced Options

The Create Web Page Wizard has several advanced options to:

  • Default extensions in case you don't want to use html as your extension
  • Allow thumbnails and extra files to be placed in a sub folders
  • Use lowercase filenames for case sensitive Unix servers
  • Remove spaces and problem characters from file names
  • Disable right click on galleries
  • Add files to allow the gallery to be auto run off a CD
  • Control sorting of gallery items
