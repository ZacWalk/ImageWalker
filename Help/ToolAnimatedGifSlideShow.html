Animated Gif Slide Show Wizard

The Animated Gif Slide Show creation wizard creates GIF files to display the images in the current folder. The wizard takes you through several steps and will prompt you for the following settings. 

Input: Allows you to select files to be processed, either all or selected images. Images in sub folders can also be processed.

Scale Size: Defines how large the animated gif image will be.

File Out: Defines the file name for the Animated Gif Slide Show and time delay between each image.

Tip: The larger the image the larger the final GIF file size. If you plan to place this image on a web page it is best to limit the size of you image.
